Women’s Fellowship Digital Bulletin

Creating an accessible, real-time community hub for the Women’s Fellowship at First Love Church. Built following WCAG accessibility guidelines with a Chrome Lighthouse score of 95.

View Live Site

Overview

The Women’s Fellowship website is a digital bulletin and community hub designed for members, many of whom are elderly and not tech-savvy. The platform enables real-time posting, access to member contact information, and easy navigation, while maintaining administrative control for moderation. The design follows WCAG guidelines and scored 95 on Chrome Lighthouse for accessibility.

The Problem

Previously, the group relied on in-person bulletins and word-of-mouth updates, which made staying informed and connected difficult. The old website could not host member posts, events, or contact information effectively. Key challenges included:

  • Posts needed categorization (Prayer Requests, Provisions Needed, Events, Other Updates)
  • Many members were elderly and unfamiliar with digital tools
  • Secure access to member phone and email information was required
  • Admin moderation was necessary, with users able to edit only their own posts

Goals

The platform aimed to:

  • Allow members to post updates in real-time
  • Provide secure access to member contact info (phone/email)
  • Be intuitive and accessible for elderly users
  • Enable backend moderation for administrators

Process

1. Research & Discovery

Informal interviews with members and leaders identified the need for a familiar bulletin board experience, simplicity, and accessibility, as well as secure display of member contact information.

2. Wireframing

Low-fidelity wireframes outlined four bulletin categories and a member directory section, mimicking the traditional bulletin board layout while simplifying navigation.

3. Visual Design

Design emphasized readability with large fonts, high contrast, persistent category navigation, and a minimal posting flow.

4. Accessibility Features

Accessibility was a primary focus, including large buttons, high contrast, clear navigation, and mobile-friendly layouts. The site was built following WCAG guidelines, and a Chrome Lighthouse audit returned a score of 95, ensuring excellent accessibility for elderly and non-tech-savvy users.

Solution

The final platform includes:

  • Real-time bulletin board with categorized posts
  • Member directory displaying phone and email information
  • Backend tools for administrator moderation
  • User post control: members can edit only their own submissions
  • Accessibility-first design: large fonts, clear buttons, mobile-friendly layouts

Impact

  • Members can stay connected digitally, even between gatherings
  • Elderly users can confidently post, edit, and access contact info
  • Group communication is faster, more organized, and easier to moderate

Reflection

This project reinforced the importance of simplicity, familiarity, and accessibility in UX. Future improvements could include digest emails or notifications to keep members informed passively.
